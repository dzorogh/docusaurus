# Настройка доступа к каталогам в файловом менеджере

Доступы по дереву каталогов распространяется **по принципу наследования** (inheritance).

:::warning[Важно]
Настройка доступов для каталогов диска доступна для пользователей с ролью: **администратор диска**.
:::

**Базовый принцип**\
Права, назначенные на родительский каталог, автоматически применяются ко всем вложенным подкаталогам и файлам — если не заданы явные исключения.

![Доступы](/img/nastroika-dostupa-k-katalogam-v-failovom-menedzhere/tree.png)

### Настройка доступа: частные правила

1. Перейдите в раздел корпортала **Диск**.
2. Выберите каталог для которого необходимо назначить частные доступы.\
   \&#xNAN;*Например: доступ, назначенный на каталог верхнего уровня, распространится на все вложенные папки.*

![Доступы](/img/nastroika-dostupa-k-katalogam-v-failovom-menedzhere/top-level-access.png)

3. В окне **Действия** **над** **текущей** **папкой** нажмите **Изменить** **доступы**.

![Доступы](/img/nastroika-dostupa-k-katalogam-v-failovom-menedzhere/create-access.png)

4. В окне **Доступы** в секции **Частный** **доступ** нажмите **+** **Добавить** **доступ**.
5. В поле выбора **Тип** выберите `Пользователь` .
6. В поле выбора **Пользователь** выберите пользователя из предложенного списка.
7. В поле выбора **Доступ** выберите нужное:
   1. `Запись`
   2. `Чтение`
   3. `Запрещён`&#x20;
8. Чтобы добавить ещё пользователя, нажмите **«+»**, для удаления существующего доступа нажмите **«-»** напротив соответствующей записи.
9. После того как сформируете актуальный список пользователей с нужными доступами нажмите **Сохранить**.

При успешном сохранении доступов в рабочей области корпортала отобразится системное сообщение `Настройки доступов сохранены`, а окне **Действия** **над** **текущей** **папкой** в секции **Доступ** отобразится информация с указанием количества заданных частных правил.
