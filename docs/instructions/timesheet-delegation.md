# Инструкция по делегированию доступов к табелям сотрудников

Пользователи с ролью должности **Руководитель** **филиала** имеют доступ к табелю сотрудников и могут делегировать свой доступ своим подчинённым через функцию делегирования.

**Для** **этого**:

1. На корпоративном портале в правом верхнем углу нажмите на иконку пользователя (ваша иконка на корпоративном портале), после выберите пункт **Личный** **кабинет**.
2. Для открытия формы **Делегирование** **доступа** выберите вкладку **Делегирование** **доступа**.
3. В форме **Делегирование** **доступа** в поле выбора **График** **работы** начните вводить данные сотрудника, которому вы ходите делегировать свой доступ, после выберите сотрудника из предложенного списка.
4. Нажмите **Сохранить**: в рабочей области раздела отобразится системное сообщение `Доступы сохранены`.

При успешном делегировании доступа у соответствующего сотрудника появится доступ к табелям сотрудников уровня руководителя филиала.
