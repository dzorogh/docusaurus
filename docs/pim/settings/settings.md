# Настройки

Раздел **Настройки** — это специализированный раздел, представляющий собой организационную структуру или информационную модель **PIM-системы**.

:::warning[Важно]
Полный функционал раздела доступен для пользователей с ролью доступа: **PIM Admin**.
:::

### Управление настройками

Для перехода в рабочую область раздела **Настройки** в системном меню портала перейдите по пунктам **Магазин** **и** **каталог** ➔ **Настройки**.

Для просмотра нужного раздела настроек на панели настроек выберите нужный раздел:

* [Категории товаров](https://globaldrive.gitbook.io/cp/modules/magazin-i-katalog/nastroiki/kategorii-tovarov)
* [Группы атрибутов](https://globaldrive.gitbook.io/cp/modules/magazin-i-katalog/nastroiki/gruppy-atributov)
* [Атрибуты](https://globaldrive.gitbook.io/cp/modules/magazin-i-katalog/nastroiki/atributy)
* [Семейства товаров](https://globaldrive.gitbook.io/cp/modules/magazin-i-katalog/nastroiki/semeistva-tovarov)
* [Метаданные](https://globaldrive.gitbook.io/cp/modules/magazin-i-katalog/nastroiki/metadannye)
* [Регионы](https://globaldrive.gitbook.io/cp/modules/magazin-i-katalog/nastroiki/regiony)
* [Производственные площадки](https://globaldrive.gitbook.io/cp/modules/magazin-i-katalog/nastroiki/proizvodstvennye-ploshadki)
