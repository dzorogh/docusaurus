# Маршруты

Существующие маршруты отображаются в рабочей области раздела в виде списка.

:::tip[Обратите внимание]
Для отображения удалённых записей включите переключатель **Показывать архивные**.
:::

## Структура списка маршрутов

| Наименование поля | Описание |
| --- | --- |
| **ID** | Уникальный идентификатор маршрута в базе данных **PIM‑системы** |
| **Имя** | Название маршрута |
| **Производственная площадка** | Код производственной площадки. В конце каждой записи отображается значок (кнопка) для редактирования (![](./assets/edit.png)) соответствующего маршрута.

## Как создать маршрут

1. Перейдите в раздел настроек **Маршруты**.
2. В рабочей области раздела нажмите **Добавить**.
3. Введите необходимые данные в форме **Новый маршрут**:
   1. Поле ввода **Имя** (обязательное): определяет название маршрута. Введите название маршрута.
   2. Поле выбора **Производственная площадка** (обязательное): позволяет выбрать код производственной площадки (начальная точка маршрута). Выберите код производственной площадки.
      :::tip[Обратите внимание]
      В одном маршруте может быть несколько производственных площадок.
      :::
   3. Поле выбора **Компания** (обязательное): позволяет выбрать существующие в системе компании. Выберите название компании.
      :::tip[Обратите внимание]
      Сущность описана в разделе **Компании**.
      :::
   4. Поле выбора **Склад компании** (обязательное): позволяет выбрать существующие склады компании (конечная точка маршрута). Выберите склад компании.
   5. Опции **Тип транспорта**:
      1. **Железная дорога**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
      2. **Грузовик**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
      3. **Море**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
      4. **Воздух**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
   6. Поле ввода **Время в пути (дней)**: определяет ориентировочное количество дней маршрута. Введите необходимое значение (дней). &#x20;
   7. Поле ввода **Стоимость за куб**: определяет ориентировочную стоимость за кубический метр в выбранной валюте. Выберите валюту и введите значение стоимости.
   8. Поле ввода **Стоимость за 40' контейнер (USD)**: определяет ориентировочную стоимость доставки при использовании 40 футового контейнера в долларах США (USD). Введите необходимое значение стоимости в долларах США (USD).
4. Поле для ввода комментария **Внутренний комментарий**: введите текст комментария.
   :::tip[Обратите внимание]
   Внутренний комментарий доступен для просмотра и редактирования в форме маршрута.
   :::
5. Нажмите **Создать**, для отмены в меню навигации нажмите **Назад**.

При успешном создании маршрута вновь созданный маршрут отобразится в списке раздела, а рабочей области портала отобразится системное сообщение: `Маршрут <Имя маршрута> создан`.

## Как отредактировать маршрут

1. Перейдите в раздел настроек **Маршруты**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. Отредактируйте необходимые данные в форме редактирования маршрута:
   1. Поле ввода **Имя** (обязательное): определяет название маршрута. Введите название маршрута.
   2. Поле выбора **Производственная площадка** (обязательное): позволяет выбрать код производственной площадки (начальная точка маршрута). Выберите код производственной площадки.
      :::tip[Обратите внимание]
      В одном маршруте может быть несколько производственных площадок.
      :::
   3. Поле выбора **Компания** (обязательное): позволяет выбрать существующие в системе компании. Выберите название компании.
      :::tip[Обратите внимание]
      Сущность описана в разделе **Компании**.
      :::
   4. Поле выбора **Склад компании** (обязательное): позволяет выбрать существующие склады компании (конечная точка маршрута). Выберите склад компании.
   5. Опции **Тип транспорта**:
      1. **Железная дорога**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
      2. **Грузовик**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
      3. **Море**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
      4. **Воздух**: установите опцию, если соответствующий тип транспорта используется в маршруте.&#x20;
   6. Поле ввода **Время в пути (дней)**: определяет ориентировочное количество дней маршрута. Введите необходимое значение (дней). &#x20;
   7. Поле ввода **Стоимость за куб**: определяет ориентировочную стоимость за кубический метр в выбранной валюте. Выберите валюту и введите значение стоимости.
   8. Поле ввода **Стоимость за 40' контейнер (USD)**: определяет ориентировочную стоимость доставки при использовании 40 футового контейнера в долларах США (USD). Введите необходимое значение стоимости в долларах США (USD).
4. Поле для ввода комментария **Внутренний комментарий**: введите текст комментария.
   :::tip[Обратите внимание]
   Внутренний комментарий доступен для просмотра и редактирования в форме маршрута.
   :::
5. Нажмите **Сохранить**, для отмены в меню навигации нажмите **Назад**.

При успешном редактировании отредактированный маршрут отобразится в списке раздела с новыми данными, а в рабочей области портала отобразится системное сообщение: `Маршрут <Имя маршрута> обновлен`.

## Как удалить маршрут

1. Перейдите в раздел настроек **Маршруты**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. В форме редактирования маршрута нажмите **Удалить**, для подтверждения удаления нажмите **Подтвердить**, для отмены действия нажмите **Отмена**.

При успешной операции удаленный маршрут исчезнет из списка раздела, а в рабочей области портала отобразится системное сообщение:  `Маршрут <Имя маршрута> удален`.

## Как восстановить маршрут

1. Перейдите в раздел настроек **Маршруты**.
2. В рабочей области раздела установите переключатель **Показать архивные**.
3. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
4. В форме редактирования маршрута нажмите **Восстановить**, для отмены в меню навигации нажмите **Назад**.

При успешной операции восстановления восстановленный маршрут отобразится в списке раздела, а в рабочей области портала отобразится системное сообщение:  `Маршрут <Имя маршрута> восстановлен`.
