# Склады поставщиков

Существующие склады поставщиков отображаются в рабочей области раздела в виде списка.

:::tip[Обратите внимание]
Для отображения удалённых записей включите переключатель **Показывать архивные**.
:::

В конце каждой записи отображается значок (кнопка) для редактирования (![](./assets/edit.png)) соответствующего маршрута.

## Как создать склад

1. Перейдите в раздел настроек **Склады поставщиков**.
2. В рабочей области раздела нажмите **Добавить**.
3. Введите необходимые данные в форме **Склад** **поставщика**:
   1. Поле ввода **Имя** (обязательное): определяет название склада. Введите название склада поставщика.
   2. Поле ввода **Код** (обязательное): определяет код склада поставщика. Введите код склада поставщика.
   3. Поле ввода **Страна** (обязательное): определяет страну склада поставщика. Введите страну склада поставщика.
   4. Поле ввода **Город** (обязательное): определяет город склада поставщика. Введите город склада поставщика.
   5. Поле ввода **Адрес** (обязательное): определяет адрес склада поставщика. Введите адрес склада поставщика.
   6. Поле с возможностью множественного выбора **Ответственный** (обязательное): выберите ответственных склада поставщика из предложенного списка.
4. Нажмите **Создать**, для отмены в меню навигации нажмите **Назад**.

При успешном создании склада поставщика вновь созданный склад отобразится в списке раздела, а рабочей области портала отобразится системное сообщение: `Склад поставщика <Имя склада поставщика> создан`.

## Как отредактировать склад

1. Перейдите в раздел настроек **Склады поставщиков**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. В форме редактирования склада поставщика отредактируйте необходимые данные:
   1. Поле ввода **Имя** (обязательное): определяет название склада. Введите название склада поставщика.
   2. Поле ввода **Код** (обязательное): определяет код склада поставщика. Введите код склада поставщика.
   3. Поле ввода **Страна** (обязательное): определяет страну склада поставщика. Введите страну склада поставщика.
   4. Поле ввода **Город** (обязательное): определяет город склада поставщика. Введите город склада поставщика.
   5. Поле ввода **Адрес** (обязательное): определяет адрес склада поставщика. Введите адрес склада поставщика.
   6. Поле с возможностью множественного выбора **Ответственный** (обязательное): выберите ответственных склада поставщика из предложенного списка.
4. Нажмите **Сохранить**, для отмены в меню навигации нажмите **Назад**.

При успешном редактировании склад с новыми данными отобразится в списке раздела, а рабочей области портала отобразится системное сообщение: `Склад поставщика <Имя склада поставщика> обновлен`.

## Как удалить склад

1. Перейдите в раздел настроек **Склады поставщиков**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. В форме редактирования склада нажмите **Удалить**, для подтверждения удаления нажмите **Подтвердить**, для отмены действия нажмите **Отмена**.

При успешной операции удаленный склад исчезнет из списка раздела, а в рабочей области портала отобразится системное сообщение:  `Склад поставщика <Имя склада поставщика> удален`.

## Как восстановить склад

1. Перейдите в раздел настроек **Склады поставщиков**.
2. В рабочей области раздела установите переключатель **Показать архивные**.
3. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
4. В форме редактирования склада нажмите **Восстановить**, для отмены в меню навигации нажмите **Назад**.

При успешной операции восстановления восстановленный склад поставщика отобразится в списке раздела, а в рабочей области портала отобразится системное сообщение:  `Склад поставщика <Имя склада поставщика> восстановлен`.
