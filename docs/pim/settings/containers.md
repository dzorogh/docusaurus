# Контейнеры

Существующие контейнеры отображаются в рабочей области раздела в виде списка.

:::tip[Обратите внимание]
* Для отображения удалённых записей включите переключатель **Показывать архивные**.
* Один из основных стандартов, регламентирующий классификацию контейнеров различных типов: **ISO 668:1995** («Series 1 freight containers — Classification, dimensions and ratings»).
:::

## Структура списка контейнеров

| Наименование поля | Описание |
| --- | --- |
| **ID** | Уникальный идентификатор контейнера в базе данных **PIM‑системы** |
| **Имя** | Название контейнера |
| **Внешние размеры** | Габаритные размеры (Длина **×** Ширина **×** Высота) контейнера (мм) |
| **Внутренние размеры** | Внутренние размеры (Длина **×** Ширина **×** Высота) контейнера (мм) |
| **Максимальный вес** | Масса брутто.&#10;&#10;**Формула расчёта** массы брутто:&#10;`масса брутто = масса нетто + масса упаковки`, где:&#10;- `масса нетто` — чистый вес продукта.&#10;- `масса упаковки` — вес контейнера, коробки или другого упаковочного материала |
| **Внутренний объём** | Внутренний объём контейнера рассчитывается как произведение трёх основных внутренних размеров.&#10;&#10;**Формула расчёта**: `V = L × W × H`, где:&#10;- `V` — объём контейнера (м³).&#10;- `L` — внутренняя длина (мм).&#10;- `W` — внутренняя ширина (мм).&#10;- `H` — внутренняя высота (мм). |

В конце каждой записи отображается значок (кнопка) для редактирования (![](./assets/edit.png)) соответствующего контейнера.

## Как создать контейнер

1. Перейдите в раздел настроек **Контейнеры**.
2. В рабочей области раздела нажмите **Добавить**.
3. Введите необходимые данные в форме **Новый контейнер**:
   1. Поле ввода **Имя** (обязательное): определяет название контейнера. Введите название контейнера.
   2. Поле ввода **Внешняя длина**: определяет значение внешней длины контейнера. Введите значение внешней длины контейнера (мм).
   3. Поле ввода **Внешняя ширина**: определяет значение внешней ширины контейнера. Введите значение внешней ширины контейнера (мм).
   4. Поле ввода **Внешняя глубина**: определяет значение внешней глубины (высоты) контейнера. Введите значение внешней глубины (высоты) контейнера (мм).&#x20;
   5. Поле ввода **Внутренняя длина**: определяет значение внутренней длины контейнера. Введите значение внутренней длины контейнера (мм).
   6. Поле ввода **Внутренняя ширина**: определяет значение внутренней ширины контейнера. Введите значение внутренней ширины контейнера (мм).
   7. Поле ввода **Внутренняя глубина**: определяет значение внутренней глубины (высоты) контейнера. Введите значение внутренней глубины (высоты) контейнера (мм).&#x20;
   8. Поле ввода **Вес пустого контейнера**: определяет значение массы тары (вес пустого контейнера). Введите значение (кг).
   9. Поле ввода **Максимальный вес**: определяет значение максимального веса (масса брутто). Введите значение (кг). &#x20;
4. Нажмите **Создать**, для отмены на панели навигации нажмите **Назад**.

При успешном создании контейнера вновь созданный контейнер отобразится в списке контейнеров раздела, а в рабочей области портала отобразится системное сообщение: `Контейнер <Имя контейнера> создан`.

## Как отредактировать контейнер

1. Перейдите в раздел настроек **Контейнеры**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. Отредактируйте необходимые данные в форме редактирования контейнера:
   1. Поле ввода **Имя** (обязательное): определяет название контейнера. Отредактируйте название контейнера.
   2. Поле ввода **Внешняя длина**: определяет значение внешней длины контейнера. Отредактируйте значение внешней длины контейнера (мм).
   3. Поле ввода **Внешняя ширина**: определяет значение внешней ширины контейнера. Отредактируйте значение внешней ширины контейнера (мм).
   4. Поле ввода **Внешняя глубина**: определяет значение внешней глубины (высоты) контейнера. Отредактируйте значение внешней глубины (высоты) контейнера (мм).&#x20;
   5. Поле ввода **Внутренняя длина**: определяет значение внутренней длины контейнера. Отредактируйте значение внутренней длины контейнера (мм).
   6. Поле ввода **Внутренняя ширина**: определяет значение внутренней ширины контейнера. Отредактируйте значение внутренней ширины контейнера (мм).
   7. Поле ввода **Внутренняя глубина**: определяет значение внутренней глубины (высоты) контейнера. Отредактируйте значение внутренней глубины (высоты) контейнера (мм).&#x20;
   8. Поле ввода **Вес пустого контейнера**: определяет значение массы тары (вес пустого контейнера). Отредактируйте значение (кг).
   9. Поле ввода **Максимальный вес**: определяет значение максимального веса (масса брутто). Отредактируйте значение (кг). &#x20;
4. Нажмите **Сохранить**, для отмены в меню навигации нажмите **Назад**.

При успешном редактировании отредактированный контейнер отобразится в списке раздела с новыми данными, а в рабочей области раздела отобразится системное сообщение: `Контейнер <Имя контейнера> обновлен`.

## Как удалить контейнер

1. Перейдите в раздел настроек **Контейнеры**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. В форме контейнера нажмите **Удалить**, для подтверждения удаления нажмите **Подтвердить**, для отмены действия нажмите **Отмена**.

При успешной операции удаленный контейнер исчезнет из списка раздела, а в рабочей области раздела отобразится системное сообщение: `Контейнер <Имя контейнера> удален`.

## Как восстановить контейнер

1. Перейдите в раздел настроек **Контейнеры**.
2. В рабочей области раздела установите переключатель **Показать архивные**.
3. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
4. В форме контейнера нажмите **Восстановить,** для отмены в меню навигации нажмите **Назад**.

При успешной операции восстановления восстановленный контейнер отобразится в списке раздела, а в рабочей области раздела отобразится системное сообщение:  `Контейнер <Имя контейнера> восстановлен`.
