# Бренды

Существующие бренды отображаются в рабочей области раздела в виде списка.

:::tip[Обратите внимание]
Для отображения удалённых записей включите переключатель **Показывать архивные**.
:::

## Структура списка брендов

| Наименование поля | Описание |
| --- | --- |
| **ID** | Уникальный идентификатор бренда в базе данных **PIM‑системы** |
| **Код** | Код бренда |
| **Имя** | Имя бренда |
| **Описание** | Описание бренда |
| **Количество товаров** | Отображает количество товаров соответствующего бренда.&#10;&#10;<div className="admonition admonition-info"><div className="admonition-heading">💡 **Для перехода** в справочник товаров, отфильтрованных по бренду, нажмите на значение поля для выбранного бренда</div></div>|

В конце каждой записи отображается значок (кнопка) для редактирования (![](./assets/edit.png)) соответствующего бренда.

## Как создать бренд

1. Перейдите в раздел настроек **Бренды**.
2. В рабочей области раздела нажмите **Добавить**.
3. Введите необходимые данные в форме **Новый бренд**:
   1. Поле ввода **Имя** (обязательное): определяет название бренда. Введите название бренда.
   2. Поле ввода **Код**: определяет название кода бренда. Введите название кода бренда.
   3. Поле ввода **Описание**: введите описание бренда.
4. Нажмите **Создать**, для отмены в меню навигации нажмите **Назад**.

При успешном создании бренда вновь созданный бренд отобразится в списке брендов раздела, а в рабочей области портала отобразится системное сообщение: `Бренд <Имя бренда> создан`.

## Как отредактировать бренд <a href="#redaktirovanie-konteinera" id="redaktirovanie-konteinera"></a>

1. Перейдите в раздел настроек **Бренды**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. Отредактируйте необходимые данные в форме бренда:
   1. Поле ввода **Имя** (обязательное): определяет название бренда. Отредактируйте название бренда.
   2. Поле ввода **Код**: определяет название кода бренда. Отредактируйте код бренда.
   3. Поле ввода **Описание**: отредактируйте описание бренда.
4. Нажмите **Сохранить**, для отмены в меню навигации нажмите **Назад**.

При успешном редактировании бренд отобразится в списке раздела с новыми данными, а в рабочей области портала отобразится системное сообщение: `Бренд <Имя бренда> обновлен` .

## Как удалить бренд

1. Перейдите в раздел настроек **Бренды**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. В форме бренда нажмите **Удалить**, для подтверждения удаления нажмите **Подтвердить**, для отмены действия нажмите **Отмена**.

При успешной операции удаленный бренд исчезнет из списка, а в рабочей области портала отобразится системное сообщение: `Бренд <Имя бренда> удален`.

## Как восстановить бренд

1. Перейдите в раздел настроек **Бренды**.
2. В рабочей области раздела установите переключатель **Показать архивные**.
3. Выберите нужную запись бренда и нажмите редактировать (![](./assets/edit.png)).
4. В форме бренда нажмите **Восстановить**.

При успешном восстановлении бренд отобразится в списке, а в рабочей области портала отобразится системное сообщение: `Бренд <Имя бренда> восстановлен`.
