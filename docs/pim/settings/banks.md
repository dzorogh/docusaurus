# Банки

Существующие банки отображаются в рабочей области раздела в виде списка.

:::tip[Обратите внимание]
* Для отображения удалённых записей включите переключатель **Показывать архивные**.
* Для поиска существующих банков начните вводить имя искомого банка в поле **Поиск**.
:::

## Структура списка банков

<table><thead><tr><th width="182">Наименование поля</th><th>Описание</th></tr></thead><tbody><tr><td><strong>ID</strong></td><td>Уникальный идентификатор банка в базе данных <strong>PIM-системы</strong></td></tr><tr><td><strong>Имя</strong></td><td>Имя банка</td></tr></tbody></table>

В конце каждой записи отображается значок (кнопка) для редактирования (![](./assets/edit.png)) соответствующего банка.

## Как создать банк

1. Перейдите в раздел настроек **Банки**.
2. В рабочей области раздела нажмите **Добавить**.
3. Введите необходимые данные в форме **Новый банк**:
   1. Поле ввода **Имя** (обязательное): определяет название банка. Введите название банка.
4. Нажмите **Создать**, для отмены в меню навигации нажмите **Назад**.

При успешном создании банка вновь созданный банк отобразится в списке, а в рабочей области портала отобразится системное сообщение: `Банк <Имя банка> создан`.

## Как отредактировать банк <a href="#redaktirovanie-konteinera" id="redaktirovanie-konteinera"></a>

1. Перейдите в раздел настроек **Банки**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. Отредактируйте необходимые данные в форме банка:
   1. Поле ввода **Имя** (обязательное): определяет название банка. Отредактируйте название банка.
4. Нажмите **Сохранить**, для отмены в меню навигации нажмите **Назад**.

При успешном редактировании банк отобразится в списке раздела с новыми данными, а в рабочей области портала отобразится системное сообщение: `Банк <Имя банка> обновлен` .

## Как удалить банк

1. Перейдите в раздел настроек **Банки**.
2. В рабочей области раздела выберите нужную запись и нажмите редактировать (![](./assets/edit.png)).
3. В форме банка нажмите **Удалить**, для подтверждения удаления нажмите **Подтвердить**, для отмены действия нажмите **Отмена**.

При успешной операции удаленный банк исчезнет из списка, а в рабочей области портала отобразится системное сообщение: `Банк <Имя банка> удален`.

## Как восстановить банк

1. Перейдите в раздел настроек **Банки**.
2. В рабочей области раздела установите переключатель **Показать архивные**.
3. Выберите нужную запись банка и нажмите редактировать (![](./assets/edit.png)).
4. В форме банка нажмите **Восстановить**.

При успешном восстановлении банк отобразится в списке, а в рабочей области портала отобразится системное сообщение: `Банк <Имя банка> восстановлен`.
