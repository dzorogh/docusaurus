# Управление проектами

Раздел **Проекты** — это специализированный раздел, предназначенный для управления проектами.

Для перехода в рабочую область раздела в системном меню портала перейдите по пунктам меню **Задачи** ➔ **Проекты**.&#x20;

| -------------------------------------- | -------------------- | ---------------- | ----------- | -------------------------------------- |
| **Редактировать участников и их роли** | Да                   | Нет              | Нет         | Нет                                    |
| **Редактировать тенанты**              | Да                   | Нет              | Нет         | Нет                                    |
| **Редактировать настройки проекта**    | Да                   | Нет              | Нет         | Нет                                    |
| **Редактировать стадии**               | Да                   | Нет              | Нет         | Нет                                    |
| **Создавать задачи**                   | Да                   | Да               | Нет         | Нет                                    |
| **Видеть проект в списках**            | Да                   | Да               | Да          | <p></p><p>Нет</p><p></p>               |
| **Видеть все задачи проекта**          | Да                   | Да               | Да          | Нет (только если есть доступ к задаче) |

## Создание проекта

1. Перейдите в раздел **Проекты**.
2. Нажмите кнопку **Добавить**.
3. Заполните форму **Создание проекта**:
   1. Поле ввода **Название**: определяет название проекта. Введите название проекта.
   2. Поле вода **Описание**: определяет описание проекта. Введите описание проекта.
   3. Поле с поддержкой выбора множества значений **Тенант**: определяет привязку проекта к выбранным тенантам. Выберите тенант.&#x20;
4. Нажмите **Сохранить**.

При успешном создании нового проекта вы будете перенаправлены в раздел проекта **Общая информация**, а в рабочей области раздела отобразится системное сообщение: `Проект создан`.

:::tip[Обратите внимание]
При необходимости отредактируйте вновь созданный проект в соответствии с описанием раздела [Редактирование проекта](#redaktirovanie-proekta).
:::

## Редактирование проекта {#redaktirovanie-proekta}

1. Перейдите в раздел **Проекты**.
2. Найдите нужную запись проекта, после нажмите **Редактировать**.
3. Заполните раздел вновь созданного проекта **Общая информация**, если ещё не заполнили, после нажмите **Сохранить**.\
   При успешном редактировании отобразится системное сообщение `Проект отредактирован`.&#x20;
4. В меню проекта выберите раздел **Пользователи** и заполните нужную информацию:
   :::tip[Обратите внимание] 
   <div data-gb-custom-block data-tag="hint" data-style="warning" class="hint hint-warning"><p>Существующие в проекте пользователи отображаются в разделе в виде списка.</p></div>
   :::
   1. Поле с поддержкой выбора множества значений **Пользователь**: позволяет выбрать существующих в системе пользователей. Выберите нужных пользователей.
   2. Поле выбора **Роль**: позволяет выбрать роль пользователя в проекте. Выберите нужную роль в соответствии с разделом [Роли пользователей в проектах](#roli-polzovatelei-v-proektakh).
   3. Нажмите кнопку **Добавить**.

    При успешном добавлении пользователей вновь добавленные пользователи отобразятся в списке пользователей раздела.

7. В меню проекта выберите раздел **Кастомные поля**, после нажмите **Добавить**:
   :::tip[Обратите внимание]
   <div data-gb-custom-block data-tag="hint" data-style="warning" class="hint hint-warning"><p>Существующие в проекте кастомные поля отображаются в разделе в виде списка.</p></div>
   :::
   1. Заполните форму **Создание кастомного поля**:
      1. Поле ввода **Название**: определяет название кастомного поля. Введите название кастомного поля.
      2. Поле выбора **Тип поля**: позволяет выбрать тип данных для кастомного поля (значение по умолчанию: **Строка**). Выберите необходимый тип данных из списка:&#x20;
         1. **Строка**: задает тип данных строка.
         2. **Число**: задает тип данных число.
         3. **Выбор**: задает тип данных поле выбора.
         4. **Мультивыбор**: задает тип данных поле с поддержкой выбора множества значений.
      3. Поле ввода **Опции**: позволяет установить значения для полей с типом данных **Выбор** и **Мультивыбор**. Введите значение опции, после нажмите **Добавить**.
        :::tip[Обратите внимание] 
         <div data-gb-custom-block data-tag="hint" data-style="warning" class="hint hint-warning"><p>Добавьте нужное количество опций для полей с соответствующим типом.</p></div>
        :::  
      4. Нажмите **Сохранить**.

   При успешном создании кастомное поле отобразится в списке кастомных полей раздела, а в рабочей области корпортала отобразится сообщение  `Проект отредактирован`.

## Удаление проекта

1. Перейдите в раздел **Проекты**.
2. Найдите нужную запись проекта, после нажмите **Редактировать**.
3. В окне проекта нажмите **Удалить**, для подтверждения удаления нажмите Подтвердить, для отмены нажмите **Отмена**.

При успешном удалении на странице проекта появится сообщение: `В корзине`.

:::tip[Обратите внимание]
При необходимости можно восстановить удалённый проект, для этого на странице проекта нажмите **Восстановить**.
:::

## Восстановление проекта

1. Перейдите в раздел **Проекты**.
2. Установите опцию **Показать удалённые**.
3. Найдите нужную запись проекта, после нажмите **Редактировать**.
4. В окне проекта нажмите **Восстановить**.

При успешном восстановлении проекта восстановленный проект появится в списке проектов раздела.   &#x20;
