# Группы атрибутов

Раздел **Группы атрибутов**  позволяет [создавать](#sozdanie-gruppy-atributov), [удалять](#udalenie-gruppy-atributov), [редактировать](#redaktirovanie-gruppy-atributov) и [восстанавливать](#vosstanovlenie-gruppy-atributov) группы атрибутов.\
\
**Группы атрибутов** позволяют группировать атрибуты товаров для удобного представления в **PIM-системе**.

Существующие группы атрибутов отображаются в рабочей области раздела в виде списка.

### Описание полей списка групп атрибутов

### Создание группы атрибутов

1. Перейдите в раздел настроек **Группы атрибутов**.
2. Нажмите кнопку **Добавить**.
3. Заполните форму **Создание группы атрибутов**:
   1. Поле ввода **Код** (необязательное): может быть использовано внешними системами для фильтрации групп атрибутов по коду. Не заполняется пользователем.
   2. Переключатель **Активно**: определяет текущий статус группы атрибутов в системе. Значение по умолчанию — «Активно». Установите переключатель в нужное положение.
   3. Дополнительная панель вкладок:
      * По умолчанию:&#x20;
        * Поле ввода **Название** (обязательное): определяет наименование группы атрибутов. Введите наименование группы атрибутов.&#x20;
      * Языковые вкладки: определяют наличие перевода наименования группы атрибутов для выбранной локали. Заполните языковые вкладки при необходимости.

При успешном создании группы атрибутов вновь созданная группа отобразится в списке групп атрибутов.

### Редактирование группы атрибутов

1. Перейдите в раздел настроек **Группы атрибутов**.
2. В рабочей области раздела найдите нужную запись и нажмите **Редактировать**..
3. Отредактируйте форму:
   1. Поле ввода **Код** (необязательное): может быть использовано внешними системами для фильтрации групп атрибутов по коду. Не заполняется пользователем.
   2. Переключатель **Активно**: определяет текущий статус группы атрибутов в системе. Значение по умолчанию — «Активно». Установите переключатель в нужное положение.
   3. Дополнительная панель вкладок:
      * По умолчанию:&#x20;
        * Поле ввода **Название** (обязательное): определяет наименование группы атрибутов. Введите наименование группы атрибутов.&#x20;
      * Языковые вкладки: определяют наличие перевода наименования группы атрибутов для выбранной локали. Отредактируйте языковые вкладки при необходимости.

При успешном редактировании отредактированная группа отобразится в списке групп атрибутов раздела.

### Удаление группы атрибутов

1. Перейдите в раздел настроек **Группы атрибутов**.
2. В рабочей области раздела найдите нужную запись и нажмите **Редактировать**.
3. Нажмите **Удалить**. Для подтверждения удаления нажмите **Подтвердить**, для отмены удаления нажмите **Отмена**.

При успешной операции удаленная группа атрибутов исчезнет из списка раздела.

### Восстановление группы атрибутов

1. Перейдите в раздел настроек **Группы атрибутов**.
2. В рабочей области раздела установите переключатель **Показать удаленные**.
3. В рабочей области раздела найдите нужную запись и нажмите **Редактировать**, после нажмите **Восстановить**.

При успешной операции восстановления восстановленная группа атрибутов отобразится в списке раздела.
