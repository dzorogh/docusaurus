# Категории товаров

Раздел **Категории товаров**  позволяет [создавать](#sozdanie-kategorii), [удалять](#udalenie-kategorii), [редактировать](#redaktirovanie-kategorii) и [восстанавливать](#vosstanovlenie-kategorii) категории товаров.

**Категории товаров** позволяют задавать многоуровневую структуру категорий (первый уровень, второй уровень и т.д.).

Существующие категории отображаются в рабочей области раздела в виде списка.

### Создание категорий

1. Перейдите в раздел настроек **Категории товаров**.
2. Нажмите кнопку **Добавить**.
3. Заполните форму **Создание категории**:
   1. Поле ввода **Порядок** (необязательное): определяет порядок следования категории в списке. Установите нужное значение с помощью кнопок  «**+**» и «**-**». Значение по умолчанию — `1`. \
    *Например: `1` .*
   2. Поле ввода **Код** (необязательное): может быть использовано внешними системами для фильтрации категорий по коду. Не заполняется пользователем.
   3. Переключатель **Активно**: определяет текущий статус категорий в системе. Значение по умолчанию — «Активно». Установите переключатель в нужное положение.
   4. Переключатель **Категория верхнего уровня**: определяет уровень вложенности категории. Значение по умолчанию — «Активно». При необходимости установите переключать в нужное состояние и заполните поле **Родительская категория**:
      1. Поле выбора **Родительская категория**: позволяет выбрать родительскую категорию для вложенной категории. Выберите значение родительской категории из предложенного списка.
   5. Дополнительная панель вкладок:
      1. По умолчанию:&#x20;
         1. Поле ввода **Название** (обязательное): определяет наименование категории. Введите наименование категории к которой относится ваш товар.&#x20;
      2. Языковые вкладки: определяют наличие перевода наименования категории для выбранной локали. Заполните языковые вкладки при необходимости.

При успешном создании категории вновь созданная категория отобразится в списке категорий раздела.

### Удаление категорий

1. Перейдите в раздел настроек **Категории товаров**.
2. В рабочей области раздела найдите нужную запись и нажмите **Редактировать**.
3. Нажмите **Удалить**. Для подтверждения удаления нажмите **Подтвердить**, для отмены удаления нажмите **Отмена**.

При успешной операции удаленная категория исчезнет из списка категорий.

### Восстановление категорий

1. Перейдите в раздел настроек **Категории товаров**.
2. В рабочей области раздела установите переключатель **Показать удаленные**.
3. В рабочей области раздела найдите нужную запись и нажмите **Редактировать**, после нажмите **Восстановить**.

При успешной операции восстановления восстановленная ктегория отобразится в списке раздела.
